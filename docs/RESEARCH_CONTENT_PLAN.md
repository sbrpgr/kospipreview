## 에이전트 실행 방법

```bash
# 환경변수 설정
export ANTHROPIC_API_KEY=your_key

# 단일 아티클 작성 (번호 지정)
python scripts/write_research_content.py 4

# 연속 작성 (4번부터 7번)
python scripts/write_research_content.py 4 5 6 7

python scripts/write_research_content.py all
```

1. 각 아티클의 Next.js 페이지가 `frontend/src/app/research/<slug>/page.tsx`에 생성된다.

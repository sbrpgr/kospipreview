## 3. 예측 알고리즘 (핵심 설계)

### 3-1. 알고리즘 설계 철학

**원칙:**
1. 단순 곱셈 공식이 아닌 **다중 선형 회귀(Multiple Linear Regression)** 기반
2. 과거 데이터로 **회귀계수를 도출**하고, 백테스트로 정확도 검증
3. VIX 수준에 따라 **밴드 폭을 동적으로 조절**
4. 모든 계수와 적중률을 투명하게 공개

### 3-2. 예측 변수 선정 근거

| 변수 | 역할 | 선정 근거 |
|---|---|---|
| **코스피200 야간선물 등락률** | 1차 핵심 | 코스피200과 결정계수 R²=0.999. 글로벌이코노믹 2026.03 실증분석에서 "야간선물 마감가 → 코스피 시초가" 회귀 공식 도출. 가장 직접적인 선도 지표 |
| **EWY 등락률** | 1차 핵심 | 미국 상장 한국 ETF. 삼성전자·SK하이닉스 등 대형주 100개로 구성. 외국인의 한국 증시 투자 심리를 실시간 반영 |
| **NDF USD/KRW 변동률** | 1차 핵심 | 원화 약세 → 외국인 매도 압력 → 코스피 하방. 환율과 코스피는 역상관 (상관계수 약 -0.7). 외국인 순매수와 직결 |
| **WTI 원유 등락률** | 2차 보조 | 한국 에너지 수입 의존도 高. 유가 급등 → 무역수지 악화 우려 → 코스피 하방 압력. 단, 정유·에너지주에는 호재 |
| **VIX (변동성 지수)** | 밴드 조절용 | 직접 예측 변수가 아닌 **밴드 폭 결정** 변수. VIX 높을수록 불확실성 ↑ → 밴드 확대 |
| **S&P 500 등락률** | 2차 보조 | 글로벌 투자심리 대리변수. 코스피와 동조화 경향 |

### 3-3. 예측 모델 구조

#### 모델 A: 회귀 기반 포인트 예측 (메인)

```
코스피_시초가_예측 = β₀ + β₁×(야간선물_등락률) 
                       + β₂×(EWY_등락률) 
                       + β₃×(NDF_환율_변동률) 
                       + β₄×(WTI_등락률)
                       + β₅×(S&P500_등락률)
```

**핵심:**
- β₀ = 절편 (전일 코스피 종가 기준으로 보정)
- β₁~β₅ = 과거 데이터 회귀분석으로 도출된 계수
- 모든 등락률은 **전일 한국장 마감 시점 대비** 변동률

**실제 구현 시 변환:**
```
코스피_시초가_예측 = 전일_코스피_종가 × (1 + 예측_등락률)

예측_등락률 = α₀ + α₁×(야간선물_등락률) 
                + α₂×(EWY_등락률) 
                + α₃×(NDF_환율_변동률)
                + α₄×(WTI_등락률)
                + α₅×(S&P500_등락률)
```

> 등락률을 종속변수로 쓰는 이유: 코스피 절대 수준이 변해도 모델이 유효하도록

#### 모델 B: 야간선물 단독 변환 (보조/크로스체크)

글로벌이코노믹의 실증분석(2025.01~2026.03 데이터) 결과:
```
예상_코스피 = 6.1948 × (코스피200_야간선물_지수) + 480.54
표준오차 = ±32포인트 (68% 신뢰구간)
R² = 0.999 (코스피 vs 코스피200 관계)
```

이 모델은 야간선물 단독으로 빠르게 계산할 수 있어 **크로스체크용**으로 병행.
모델 A와 모델 B 결과가 크게 괴리하면 신뢰도를 낮춤.

### 3-4. 밴드(범위) 산출

단순 포인트 예측은 의미 없음. **확률적 밴드**가 핵심.

```
밴드_상단 = 포인트_예측값 + (동적_표준오차 × 밴드_배수)
밴드_하단 = 포인트_예측값 - (동적_표준오차 × 밴드_배수)
```

**동적 표준오차:**
- 기본값: 과거 N일간 예측 잔차(residual)의 표준편차
- 초기값 (백테스트 전): ±0.3% (약 8포인트 @코스피 2700 기준)

**밴드 배수 (VIX 기반 동적 조절):**

| VIX 수준 | 밴드 배수 | 의미 |
|---|---|---|
| VIX < 15 | ×1.0 | 저변동성. 좁은 밴드 |
| 15 ≤ VIX < 20 | ×1.0 | 보통. 기본 밴드 |
| 20 ≤ VIX < 25 | ×1.3 | 불안 확대. 밴드 30% 확대 |
| 25 ≤ VIX < 30 | ×1.5 | 공포 구간. 밴드 50% 확대 |
| VIX ≥ 30 | ×2.0 | 극단적 변동성. 밴드 2배 |

### 3-6. 백테스트 및 모델 검증 계획

#### Phase 1: 초기 계수 도출 (서비스 출시 전)

```
1. 과거 데이터 수집 (최소 1년, 이상적으로 2~3년)
   - 코스피 일별 시초가, 종가
   - 코스피200 야간선물 마감가
   - EWY 종가 (미국장 마감 기준)
   - NDF USD/KRW (미국장 마감 시점)
   - WTI 종가
   - S&P500 종가
   - VIX 종가

2. 전처리
   - 각 변수를 "전일 한국장 마감 대비 등락률"로 변환
   - 종속변수: 당일 코스피 시초가 등락률 (전일 종가 대비)
   - 결측치·이상치 처리 (±3σ 이상 제거 또는 윈저화)

3. 회귀분석
   - OLS(최소제곱법) 다중 회귀
   - 각 변수의 회귀계수(β), p-value, R² 확인
   - p-value > 0.05인 변수 제거 (통계적 유의성 없는 변수)
   - 다중공선성 체크 (VIF < 5)
   - 최종 모델의 Adjusted R² 확인

4. 교차검증
   - 데이터를 훈련(80%) / 테스트(20%)로 분리
   - 또는 롤링 윈도우 방식 (최근 120일로 학습 → 다음 1일 예측 → 이동)
   - MAE(평균절대오차), RMSE(평균제곱근오차) 산출
```

#### Phase 2: 백테스트 적중률 산출

```
적중 판정 기준:
- "밴드 내 적중": 실제 시초가가 예측 밴드(상단~하단) 안에 들어왔는지
- "방향 적중": 예측 등락 방향(상승/하락)이 실제와 일치했는지

목표 적중률:
- 밴드 내 적중: 70% 이상 (68% 신뢰구간 기준이면 68%가 이론적 기대값)
- 방향 적중: 75% 이상

보고 지표:
- 최근 30일 밴드 적중률
- 최근 30일 방향 적중률  
- 최근 30일 MAE (평균절대오차, 포인트)
- 전체 기간 적중률
```

#### Phase 3: 모델 자동 재학습 (운영 중)

```
- 매주 일요일 자동으로 최근 120거래일 데이터로 회귀계수 재계산
- 재계산된 계수가 기존 대비 크게 변하면 관리자 알림
- 적중률이 60% 아래로 떨어지면 알림 → 수동 점검
- 재학습 결과는 로그에 기록 (계수 변화 추적)
```

### 3-7. 예측 알고리즘 의사코드

```python
def calculate_prediction(data, coefficients, prev_kospi_close):
    """
    data: {
        night_futures_change: float,  # 야간선물 등락률 (%)
        ewy_change: float,            # EWY 등락률 (%)
        ndf_change: float,            # NDF 환율 변동률 (%)
        wti_change: float,            # WTI 등락률 (%)
        sp500_change: float,          # S&P500 등락률 (%)
        vix: float,                   # VIX 현재값
        night_futures_price: float,   # 야간선물 현재 포인트
    }
    coefficients: {α₀, α₁, α₂, α₃, α₄, α₅, residual_std}
    """
    
    # ── 모델 A: 다중 회귀 ──
    predicted_change = (
        coefficients.α₀
        + coefficients.α₁ * data.night_futures_change
        + coefficients.α₂ * data.ewy_change
        + coefficients.α₃ * data.ndf_change
        + coefficients.α₄ * data.wti_change
        + coefficients.α₅ * data.sp500_change
    )
    
    point_prediction_A = prev_kospi_close * (1 + predicted_change / 100)
    
    # ── 모델 B: 야간선물 단독 변환 (크로스체크) ──
    point_prediction_B = 6.1948 * data.night_futures_price + 480.54
    
    # ── 최종 포인트 예측: 모델 A 우선, B로 검증 ──
    divergence = abs(point_prediction_A - point_prediction_B) / prev_kospi_close * 100
    
    if divergence < 0.5:  # 괴리 0.5% 미만이면 A 채택
        point_prediction = point_prediction_A
    else:
        # 괴리 크면 가중 평균 (A:70%, B:30%)
        point_prediction = point_prediction_A * 0.7 + point_prediction_B * 0.3
    
    # ── 밴드 산출 ──
    base_std = coefficients.residual_std  # 과거 잔차의 표준편차
    
    # VIX 기반 밴드 배수
    if data.vix < 20:
        band_multiplier = 1.0
    elif data.vix < 25:
        band_multiplier = 1.3
    elif data.vix < 30:
        band_multiplier = 1.5
    else:
        band_multiplier = 2.0
    
    band_width = base_std * band_multiplier
    band_upper = point_prediction + band_width
    band_lower = point_prediction - band_width
    
    # ── 신뢰도 등급 ──
    directions_agree = (
        sign(data.night_futures_change) == sign(data.ewy_change)
    )
    all_agree = directions_agree and (
        sign(data.night_futures_change) == sign(data.sp500_change)
    )
    
    if all_agree and data.vix < 20 and divergence < 0.2:
        confidence = 5
    elif directions_agree and divergence < 0.3:
        confidence = 4
    elif data.vix < 25:
        confidence = 3
    elif data.vix < 30:
        confidence = 2
    else:
        confidence = 1
    
    return {
        'point_prediction': round(point_prediction, 2),
        'band_upper': round(band_upper, 2),
        'band_lower': round(band_lower, 2),
        'predicted_change_pct': round(predicted_change, 2),
        'confidence': confidence,
        'model_a': round(point_prediction_A, 2),
        'model_b': round(point_prediction_B, 2),
        'divergence_pct': round(divergence, 2),
        'vix': data.vix,
        'band_multiplier': band_multiplier,
        'last_updated': datetime.now()
    }
```

### 3-8. 변수별 예상 영향 방향 (사전 가설)

| 변수 | 코스피 시초가에 대한 예상 부호 | 설명 |
|---|---|---|
| 야간선물 등락률 | **+ (양)** | 야간선물 상승 → 코스피 상승 출발. 가장 직접적 |
| EWY 등락률 | **+ (양)** | EWY 상승 → 한국 대형주 상승 심리 |
| NDF 환율 변동률 | **- (음)** | 원화 약세(환율↑) → 외국인 매도 → 코스피 하방 |
| WTI 등락률 | **±** | 소폭 상승은 경기회복 시그널(+), 급등은 비용 부담(-). 비선형 가능 |
| S&P500 등락률 | **+ (양)** | 글로벌 위험선호 지표 |

> 회귀분석 결과 부호가 가설과 다르거나 유의하지 않은 변수는 제거.
> WTI가 통계적으로 유의하지 않으면 모델에서 제외 (과적합 방지).

### 4-3. 1분 간격 수집 시 고려사항

```
- Yahoo Finance API: 무료 티어에서 분당 요청 수 제한 확인 필요
  → 제한 걸리면 5초 간격 배치 요청으로 변경
  → 또는 yfinance 파이썬 패키지 사용 (비공식이지만 제한 느슨)
  
- Investing.com 스크래핑: 1분 간격이면 IP 차단 위험
  → Playwright 헤드리스 브라우저 + 랜덤 User-Agent
  → 차단 시 AI 크롤링 백업으로 전환
  → 또는 TradingView 위젯의 데이터를 서버사이드에서 읽는 방식 검토

- 서버 부하: 12시간 × 60분 = 720회/일 수집
  → 각 수집에 10~15개 지표 = 최대 약 10,000 API 호출/일
  → Cloudflare Workers 무료 티어(10만 요청/일)로 충분
```

### 4-4. 데이터 수집 방식 우선순위

```
[1순위] Yahoo Finance API (yfinance)
  → EWY, KORU, SPY(S&P500), QQQ(나스닥), VIX, WTI, 금, US10Y, SOX
  → 무료, 안정적, 대부분의 미국 상장 지표 커버

[2순위] 웹 스크래핑 (Playwright/Puppeteer)
  → 야간선물 (Investing.com)
  → NDF 환율 (Investing.com)
  → 코스피 종가 (네이버 금융)

[3순위] AI 크롤링 백업
  → 1~2순위 소스 장애 시 Claude API로 페이지 내용 파싱
  → 비용 발생하므로 평상시에는 미사용

[4순위] TradingView 위젯
  → 차트 표시용으로만 사용
  → 데이터 계산에는 미사용 (위젯에서 수치 추출 어려움)
```

## 9. 바이브 코딩 작업 순서

> 아래 Step을 순서대로 AI 코딩 도구에 프롬프트로 전달.

### Step 1: 백테스트 & 회귀계수 도출
```
"코스피 시초가 예측을 위한 백테스트 스크립트를 만들어줘.

수집할 과거 데이터 (최소 1년):
- 코스피 일별 시초가, 종가 (Yahoo Finance: ^KS11)
- 코스피200 지수 (^KS200) 
- EWY 종가
- NDF USD/KRW (또는 USD/KRW 환율)
- WTI 원유 (CL=F 또는 USO)
- S&P500 (^GSPC)
- VIX (^VIX)

처리:
1. 각 변수를 '전일 한국장 마감 대비 등락률'로 변환
2. 종속변수: 당일 코스피 시초가의 전일 종가 대비 등락률
3. OLS 다중 회귀 실행
4. 각 계수의 값, p-value, R², Adjusted R² 출력
5. p-value > 0.05인 변수는 제거하고 재실행
6. VIF(다중공선성) 체크
7. 롤링 윈도우(120일) 교차검증으로 MAE, RMSE 산출
8. 밴드 적중률 산출 (±1σ 범위 내 실제 시초가 비율)
9. 결과를 coefficients.json으로 저장

야간선물 데이터가 없으면 EWY+환율+S&P500+WTI로 먼저 모델을 만들고,
야간선물 데이터를 구할 수 있으면 추가해서 모델 성능 비교.
결과 보고서를 BACKTEST_RESULTS.md로 생성."
```

### Step 2: 프로젝트 초기화
```
"코스피 시초가 예측 대시보드 프로젝트를 초기화해줘.

SSR/SSG 지원하는 프레임워크 사용.
모바일 퍼스트 반응형. 다크모드 지원.
README.md에 프로젝트 개요 작성."
```

### Step 3: 히어로 섹션 (예측 시초가)
```
"메인 페이지 최상단에 예측 코스피 시초가 밴드를 보여주는 히어로 섹션.
- 예측 범위 (예: 2,687 ~ 2,703)
- 중심 예측값 + 전일 대비 등락률
- 모델 신뢰도 (별 5개)
- 마지막 계산 시각
- 📋 복사 버튼 (클릭 시 '내일 코스피 예상 시초가: 2,687~2,703 (+0.6%), 신뢰도 4/5' 텍스트 복사)
- 어제 예측 vs 실제 결과 한 줄 표시
- 상승 빨간색, 하락 파란색 (한국 주식 컨벤션)
prediction.json에서 fetch."
```

### Step 4: 지표 카드 그리드
